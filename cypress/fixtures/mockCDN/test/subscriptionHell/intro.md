# Subscription Hell Test

This test cycles through many breakout room configurations to stress-test video/audio track subscriptions.

**What to do:**
1. Open browser console (F12) before starting
2. At each stage, verify you can see/hear the right people
3. Check console logs for subscription status
4. Note any failures

**4 participants required.** Each stage runs for 5 minutes.
