# Setup

There are now two recommended setup paths, depending on what you want to do.

## Setup for Researchers (recommended)

If you want to **run experiments locally** (design/test treatments, run pilot sessions, export data) without setting up a local Node/Empirica toolchain, use:

- **Setup for Researchers (Recommended)**

## Setup for Tool Developers

If you want to **modify the Deliberation Lab codebase** and run the full local development stack, use:

- **Setup for Tool Developers**
