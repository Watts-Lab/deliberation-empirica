# Deliberation Lab Documentation

Welcome to the Deliberation Lab researcher documentation. Use the navigation on the left to walk through the full lifecycle:

- **Getting Started** explains what the platform is and how to set up your environment.
- **Study Design** covers treatments, prompts, layouts, surveys, and conditions.
- **Running Sessions** walks through batch configuration, preflight checks, and deployment.
- **Analyzing Results** documents data exports, schemas, and analysis playbooks.
- **Developer Resources** highlights roadmap planning and contributor notes.

If you are new, begin with [Getting Started/Overview](getting-started/overview.md). For questions or bug reports, [open an issue](https://github.com/Watts-Lab/deliberation-empirica/issues).
